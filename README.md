# QuickStart Dev
Built for the **IBM Bob Dev Day Hackathon**.

## Overview
A Flask-based Data Agent that leverages **IBM Bob** for development and **watsonx.ai (Granite-3-8b)** for automated CSV diagnostics.

## Tech Stack
* **AI Pair Programmer:** IBM Bob
* **LLM:** IBM Granite-3-8b-instruct
* **Backend:** Flask / Python
* **Data Handling:** Pandas
